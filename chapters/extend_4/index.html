Color Converter Used: HTML Color Codes

  • Hexadecimal: Where rr(red), gg(green), bb(blue) are hexadecimal values between 00 and ff (same as decimal 0-255).
  • RGB: Each parameter (red, green, blue) defines the inensity of the color between 0 and 255.
  • HSL: Hue is a degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue.

Preferred color code format: I use the hexadecimal code format since it is easier to type and to remember.

HWB summary: Is a combination of Hue, Whiteness, Blackness.

CMYK summary: Is a combination of Cyan, Magenta, Yellow, and Black.
